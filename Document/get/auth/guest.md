# 获取登录用户资料

[TOC]

## 1. API 描述：

获取当前登录用户的资料。

## 2. 调用方法

> GET /api/auth/guset

## 3. 请求参数

无

## 4. 响应状态

状态码 | 说明
:---:|:---
200 | 操作成功

## 5. 响应数据

```json
{
    "code": 200,
    "message": "操作成功",
    "data": {
        "member_id": 2,		// 用户 ID
        "updated_at": "2017-07-06 22:08:16",   // 用户更新时间
        "created_at": "2017-07-06 22:08:16"     // 用户创建时间
    }
}
```

## 6. 示例

> GET /api/auth/guset
