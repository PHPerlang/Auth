# 注销登录

[TOC]

## 1. API 描述：

注销用户登录信息，注意项目统一 `X-App-Id`，不同终端维护各自的登录态。

## 2. 调用方法

> GET /api/auth/logout

## 2. 请求参数

无

## 3. 响应状态

状态码 | 说明
:---:|:---
200 | 操作成功


## 4. 响应数据

```json
{
    "code": 200,
    "message": "操作成功",
    "data": {}
}
```

## 5. 示例

> GET /api/auth/logout

**请求数据：**

无
